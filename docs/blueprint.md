# **App Name**: TaskFlow

## Core Features:

- Employee Directory: Comprehensive directory of all employees with contact information and roles.
- Leave Request Management: Allow employees to submit leave requests, and managers to approve or deny them.
- Performance Review System: Enable managers to conduct and record performance reviews for their direct reports.
- Absence Analyzer: AI-powered tool that uses historical absence data to suggest future staffing needs, considering seasonality and trends.
- Document Repository: Centralized repository for storing and sharing important company documents.
- Employee Onboarding: Guides HR through each step of onboarding.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trustworthiness and stability.
- Background color: Light blue-gray (#E8EAF6) to complement the primary color, but desaturated.
- Accent color: Purple (#7E57C2), an analogous color to the primary with different saturation and brightness to give clear contrast to the background.
- Body and headline font: 'Inter', a grotesque-style sans-serif suitable for both headlines and body text
- Use clean and professional icons from a consistent set, with visual metaphors that are self-explanatory.
- Use a card-based layout to organize information and make it easy to scan.
- Subtle transitions and animations to provide feedback and improve user experience.
